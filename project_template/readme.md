## simple command line

- init the mod
```shell
go mod init template
```

- go mod tidy
```shell
go mod tidy
```

- gofmt
```shell
gofmt -w .
```

- go run
```shell
go run main.go
```

- go build
```shell
go build main.go
```

- go test
```shell
go test ./...
```

- go test -v
```shell
go test -v ./...
```
